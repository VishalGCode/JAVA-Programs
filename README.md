# Java Programming Notes

This repository contains my Java programming notes and practice programs covering both the fundamentals of Java and Object-Oriented Programming (OOP) concepts. The programs are organized topic-wise and are intended to provide a clear understanding of Java syntax, core concepts, and their practical implementation.

## Topics Covered

### Java Basics

* Variables and Data Types
* Operators
* Input and Output
* Conditional Statements (`if`, `if-else`, `switch`)
* Loops (`for`, `while`, `do-while`)
* Arrays
* Strings
* Methods
* Constructors
* Static Members
* Command Line Arguments

### Object-Oriented Programming (OOP)

* Classes and Objects
* Encapsulation
* Inheritance
* Polymorphism
* Abstraction
* Interfaces
* Method Overloading
* Method Overriding
* Constructors and Constructor Overloading
* `this` and `super` Keywords

### Additional Concepts

* Packages
* Exception Handling
* File Handling
* Collections Framework
* Multithreading
* Java 8 Features (where applicable)

## Repository Purpose

The purpose of this repository is to:

* Strengthen my understanding of Java programming.
* Practice object-oriented programming concepts through code.
* Maintain organized, runnable Java examples for revision.
* Build a reference repository for academic learning and interview preparation.

Each program focuses on a specific concept and is written in a simple, readable manner to make learning and revision easier.

Happy Coding! 🚀
